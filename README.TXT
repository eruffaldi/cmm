Progetto di compilator C-- per il corso
di Informatica Teorica 1998/1999
by Ruffaldi Emanuele (C) 1999

Funzionalita':
* funzioni
* float, char, int, array e puntatori di ogni tipo
* non ci sono le strutture
* istruzioni condizionali
* special boolean expression evaluation
* stampa dell'albero di parsing in XML
* genera codice per assemblatore x86 
* genera codice per assemblatore Alpha AXP


Compilare con:
Visual C++ 5.0 o 6.0, oppure un compilatore C++ che supporti a pieno
i templates
